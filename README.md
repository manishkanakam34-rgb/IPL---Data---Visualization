#  IPL Data Analysis Dashboard (2008–2024)

##  Project Overview
This project is an interactive Power BI dashboard built using IPL datasets from 2008–2024. It provides insights into team performance, player statistics, venue analysis, and head-to-head records using interactive visualizations and filters.

---

##  Features

- 📊 Dashboard Overview
- 🏆 Team Analysis
- 👨‍💻 Player Analysis
- 🏟️ Venue Analysis
- ⚔️ Head-to-Head Team Analysis
- 🎯 Interactive Filters and Slicers
- 📈 KPI Cards and Dynamic Charts

---

##  Tools Used

- Power BI

- DAX
- Microsoft Excel
- CSV Dataset

##  Key Insights

- Top run scorers and wicket takers
- Best batting and bowling performances
- Venue-wise match analysis

- Interactive dashboard with dynamic filters
